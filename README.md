# 📚 Ejercicio 6 – Diseño de Base de Datos (Biblioteca)

## 📌 Descripción
Diseño e implementación de una base de datos relacional
para gestionar una biblioteca con autores, categorías,
libros y préstamos.

## 🗄️ Tablas creadas
- **autores** → datos de los autores
- **categorias** → tipos de libros
- **libros** → catálogo con relación a autores y categorías
- **prestamos** → registro de préstamos de libros

## 🔑 Conceptos practicados
- Creación de base de datos y tablas
- Claves primarias (PRIMARY KEY)
- Claves foráneas (FOREIGN KEY)
- Restricciones (NOT NULL, UNIQUE, DEFAULT)
- Inserción de datos (INSERT INTO)
- Normalización 3FN

## 🚀 Cómo ejecutar
1. Abre MySQL Workbench
2. Ejecuta el archivo biblioteca_db.sql
3. Verifica las tablas en el panel SCHEMAS

## 🛠️ Tecnologías
- MySQL 8.0
